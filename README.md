# Playwright BDD Automation Framework (TypeScript)

This project is a Playwright-based automated testing framework using Cucumber-style BDD, TypeScript, and the Page Object Model (POM).  
It automates end-to-end functional scenarios on the demo site https://www.saucedemo.com as mentioned in assignment.


 Features Implemented

 Login Scenarios  
- Valid login  
- Invalid login (error validation)

Cart & Checkout  
- Add multiple products  
- Validate cart items  
- Complete checkout flow

Sorting  
- Sort products by Name (A → Z)
- Sort products by Price (Low → Hig 
- Validation of sorted results

  how to run:

  npx bddgen && npx playwright test
